# Vintage Archive 운영 대시보드

빈티지샵 운영을 위한 개인용 웹앱입니다.

## 포함 기능
- 상품 등록 / 수정용 기본 관리
- localStorage 자동 저장
- CSV 내보내기
- JSON 백업 저장 / 불러오기
- 판매글 자동 생성
- 스토리 문구 자동 생성
- 매입가 기준 계산
- 오늘의 운영 체크리스트

## 실행 방법
```bash
npm install
npm run dev
```

## 배포 방법 (Vercel)
1. 이 폴더를 압축 해제
2. GitHub에 업로드하거나 Vercel에 직접 import
3. Framework Preset은 Vite로 자동 인식
4. Deploy 클릭
5. 생성된 링크로 접속

## 빌드 확인
```bash
npm run build
```
